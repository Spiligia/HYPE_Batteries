To run these files you will need the latest version of Kicad PCB design software (6.0.x). 
It is free and easily downloadable from https://www.kicad.org/

to open the schematic you will open the temp_proto_v2.1 (KiCad Project 6.0)
to open the pcb file you will open the temp_proto_v2.1 (KiCad Board)

to order a pcb you can either

1. zip this folder and morder from JLCpcb (all 'cam' files in this folder are made w/ respect to jlc's parameters)
2. find the setting in kicad and create new gerbers
